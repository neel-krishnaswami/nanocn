Elaboration Derivation

Setup. Assume:
  • $\text{sort } D() = \{T : () \mid F : ()\} \in \Sig$
  • $\spec\; x : D \in \Gam$
Goal. Derive: $$\Sig;\Gam \vdash \mathbf{case}\; x \;\mathbf{of}\; \{T() \bmark (x, F()) \mid z \bmark (z, T())\} \cmark (D, D) \emark \texttt{?}$$

1. Top Level: $\SpecChkCase$

SpecChk_case
(D1) $\;\Sig;\Gam \vdash x \smark D \emark x$
$M = T() : D \bmark [\cdot][\cdot]\, (x, F()) \;\mid\; z : D \bmark [\cdot][\cdot]\, (z, T())$
(D2) $\;\Sig;\Gam \vdash y, \cdot;\; M \cmark (D,D) \emark \mathit{ce}$
$$\Sig;\Gam \vdash \mathbf{case}\; x \;\mathbf{of}\; \{T() \bmark (x,F()) \mid z \bmark (z,T())\} \cmark (D,D) \emark \mathbf{let}\; y = x;\; \mathit{ce}$$

2. Scrutinee Synthesis

(D1) SpecSyn_var
$\spec\; x : D \in \Gam$
$\Sig;\Gam \vdash x \smark D \emark x$

3. Coverage: $\CovCon$

The matrix $M$ has a constructor in the leading column ($T()$), so $\CovCon$ applies.

(D2) Cov_con
$M\;\mathsf{has\_con}$   (by $\HChere$, since $T()$ is a constructor pattern)
$\text{sort } D() = \{T : () \mid F : ()\} \in \Sig$
$T : () \;\mathbf{in}\; D \;\mathbf{in}\; \Sig \quad F : () \;\mathbf{in}\; D \;\mathbf{in}\; \Sig$
(D3) $\;M / T : () \emark M_T$
(D4) $\;M / F : () \emark M_F$
(D5) $\;\Sig;\Gam \vdash x_T, \cdot;\; M_T \cmark (D,D) \emark \mathit{ce}_T$
(D7) $\;\Sig;\Gam \vdash x_F, \cdot;\; M_F \cmark (D,D) \emark \mathit{ce}_F$
$$\Sig;\Gam \vdash y, \cdot;\; M \cmark (D,D) \emark \mathbf{case}\; y\; \mathbf{of}\;\{T\; x_T \bmark \mathit{ce}_T \mid F\; x_F \bmark \mathit{ce}_F\}$$

4. Constructor Specialization

4a. $M / T : ()$

(D3) spec_con $M / T : ()$
Branch 1: $T() : D \bmark [\cdot][\cdot]\, (x,F())$
$\SCmatch$: $T$ matches $T$. Strip constructor, keep $\E$.
Result: $() : () \bmark [\cdot][\cdot]\, (x,F())$
Branch 2: $z : D \bmark [\cdot][\cdot]\, (z,T())$
$\SCvar$: $z$ is a variable. Introduce fresh $w$. Extend $\Gam'$ with $\spec\; z : D$. Extend $\E$ with $\mathbf{let}\; z = T\,w$.
Result: $w : () \bmark [\spec\; z : D][\mathbf{let}\; z = T\,w;\; \cdot]\, (z,T())$
$$M_T \;=\; () : () \bmark [\cdot][\cdot]\, (x,F()) \;\;\mid\;\; w : () \bmark [\spec\; z\!:\!D][\mathbf{let}\; z = T\,w;\; \cdot]\, (z,T())$$

4b. $M / F : ()$

(D4) spec_con $M / F : ()$
Branch 1: $T() : D \bmark [\cdot][\cdot]\, (x,F())$
$\SCskip$: $T \neq F$. Branch dropped.
Branch 2: $z : D \bmark [\cdot][\cdot]\, (z,T())$
$\SCvar$: $z$ is a variable. Introduce fresh $w'$. Extend $\Gam'$ with $\spec\; z : D$. Extend $\E$ with $\mathbf{let}\; z = F\,w'$.
Result: $w' : () \bmark [\spec\; z : D][\mathbf{let}\; z = F\,w';\; \cdot]\, (z,T())$
$$M_F \;=\; w' : () \bmark [\spec\; z\!:\!D][\mathbf{let}\; z = F\,w';\; \cdot]\, (z,T())$$

5. T-Branch: $\CovTup$ then $\CovDone$

$M_T$ has a tuple $()$ in the leading column, so $\CovTup$ applies.

5a. expand_tup $M_T$

expand_tup $M_T / \text{tup}$
Branch 1: $() : () \bmark [\cdot][\cdot]\, (x,F())$
$\ETtuple$ with $k=0$: expand 0-tuple, no new columns.
Result: $\cdot \bmark [\cdot][\cdot]\, (x,F())$
Branch 2: $w : () \bmark [\spec\; z\!:\!D][\mathbf{let}\; z = T\,w;\;\cdot]\, (z,T())$
$\ETvar$ with $k=0$: $w$ is a variable of type $()$. Expand, adding $\spec\; w : ()$ to $\Gam'$ and $\mathbf{let}\; w = ()$ to $\E$.
Result: $\cdot \bmark [\spec\; z\!:\!D,\; \spec\; w\!:\!()][\mathbf{let}\; w = ();\; \mathbf{let}\; z = T\,w;\; \cdot]\, (z,T())$
$$M'_T \;=\; \cdot \bmark [\cdot][\cdot]\, (x,F()) \;\;\mid\;\; \cdot \bmark [\spec\; z\!:\!D,\; \spec\; w\!:\!()][\mathbf{let}\; w = ();\; \mathbf{let}\; z = T\,w;\; \cdot]\, (z,T())$$

5b. $\CovTup$ output

(D5) Cov_tup
$M_T\;\mathsf{has\_tup}$   (by $\textsc{HT\_here}$)
$M_T / \text{tup} \emark M'_T$   (above)
(D6) $\;\Sig;\Gam \vdash \cdot;\; M'_T \cmark (D,D) \emark (x, F\,())$
$$\Sig;\Gam \vdash x_T, \cdot;\; M_T \cmark (D,D) \emark \underbrace{\mathbf{let}\;() = x_T;\; (x,\,F\,())}_{\mathit{ce}_T}$$

5c. $\CovDone$ for T-branch

(D6) Cov_done
$\Gam' = \cdot$, $\;\E = [\cdot]$   (first branch of $M'_T$)
$\Sig;\Gam \vdash (x,\,F\,()) \cmark (D,D) \emark (x,\, F\,())$
by $\SpecChkTuple$: $\;\Sig;\Gam \vdash x \cmark D \emark x$ ($\SpecChkSub + \SpecSynVar$) and $\;\Sig;\Gam \vdash F\,() \cmark D \emark F\,()$ ($\SpecChkInject + \SpecChkTuple_{k=0}$)
$\mathit{ce} = [\cdot][(x,\,F\,())] = (x,\, F\,())$   (by $\FillHole$)
$\Sig;\Gam \vdash \cdot;\; M'_T \cmark (D,D) \emark (x,\, F\,())$

6. F-Branch: $\CovVar$ then $\CovDone$

$M_F$ has only a variable ($w'$) in the leading column, so $\CovVar$ applies.

6a. strip_var with scrutinee $x_F$

strip_var $M_F / \text{var}\; x_F$
Branch: $w' : () \bmark [\spec\; z\!:\!D][\mathbf{let}\; z = F\,w';\; \cdot]\, (z,T())$
$\SVcons$: $w'$ is a variable with scrutinee $x_F$. Extend $\Gam'$ with $\spec\; w' : ()$. Extend $\E$ with $\mathbf{let}\; w' = x_F$.
Result: $\cdot \bmark [\spec\; z\!:\!D,\; \spec\; w'\!:\!()][\mathbf{let}\; w' = x_F;\; \mathbf{let}\; z = F\,w';\; \cdot]\, (z,T())$
$$M'_F \;=\; \cdot \bmark [\spec\; z\!:\!D,\; \spec\; w'\!:\!()][\mathbf{let}\; w' = x_F;\; \mathbf{let}\; z = F\,w';\; \cdot]\, (z,T())$$

6b. $\CovVar$ output

(D7) Cov_var
$M_F / \text{var}\; x_F \emark M'_F$   (above)
(D8) $\;\Sig;\Gam \vdash \cdot;\; M'_F \cmark (D,D) \emark \mathit{ce}_F$
$\Sig;\Gam \vdash x_F, \cdot;\; M_F \cmark (D,D) \emark \mathit{ce}_F$

6c. $\CovDone$ for F-branch

(D8) Cov_done
$\Gam' = \spec\; z\!:\!D,\; \spec\; w'\!:\!()$
$\E = \mathbf{let}\; w' = x_F;\; \mathbf{let}\; z = F\,w';\; [\cdot]$
$\Sig;\Gam, \Gam' \vdash (z,\,T\,()) \cmark (D,D) \emark (z,\, T\,())$
by $\SpecChkTuple$: $\;\Sig;\Gam,\Gam' \vdash z \cmark D \emark z$ ($\SpecChkSub + \SpecSynVar$, since $\spec\; z\!:\!D \in \Gam'$) and $\;\Sig;\Gam,\Gam' \vdash T\,() \cmark D \emark T\,()$
(D9) $\;\mathit{ce}_F = \E[(z,\, T\,())]$
$\Sig;\Gam \vdash \cdot;\; M'_F \cmark (D,D) \emark \mathit{ce}_F$

6d. Filling the evaluation context

(D9) Fill
$$\E = \mathbf{let}\; w' = x_F;\; \mathbf{let}\; z = F\,w';\; [\cdot]$$
$\FillLet$: $\;\E[(z,T\,())] = \mathbf{let}\; w' = x_F;\; (\mathbf{let}\; z = F\,w';\; [\cdot])[(z,T\,())]$
$\FillLet$: $\;= \mathbf{let}\; w' = x_F;\; \mathbf{let}\; z = F\,w';\; [\cdot][(z,T\,())]$
$\FillHole$: $\;= \mathbf{let}\; w' = x_F;\; \mathbf{let}\; z = F\,w';\; (z,T\,())$
$$\mathit{ce}_F \;=\; \mathbf{let}\; w' = x_F;\;\mathbf{let}\; z = F\,w';\; (z,\, T\,())$$
Scoping is correct: $w'$ is bound before $z = F\,w'$ references it.

7. Final Result

Assembling from $\SpecChkCase$ (Section 1), $\CovCon$ (Section 3), the T-branch (Section 5), and the F-branch (Section 6):

$$\Sig;\Gam \vdash \mathbf{case}\; x \;\mathbf{of}\; \{T() \bmark (x, F()) \mid z \bmark (z, T())\} \cmark (D,D)$$ $$\emark\; \mathbf{let}\; y = x;\; \mathbf{case}\; y\; \mathbf{of}\; \left\{\begin{array}{l} T\; x_T \bmark \mathbf{let}\; () = x_T;\; (x,\; F\,()) \\ F\; x_F \bmark \mathbf{let}\; w' = x_F;\; \mathbf{let}\; z = F\,w';\; (z,\; T\,()) \end{array}\right\}$$

After simplifying administrative redexes ($\mathbf{let}\;y = x$, $\;\mathbf{let}\;() = x_T$, $\;\mathbf{let}\;w' = x_F$):

$$\mathbf{case}\; x\; \mathbf{of}\; \left\{\begin{array}{l} T\; \_ \bmark (x,\; F\,()) \\ F\; u \bmark (F\,u,\; T\,()) \end{array}\right\}$$

In the T-branch, $x$ refers to the outer variable. In the F-branch, the pattern variable $z$ was bound to $F\,w' = F\,u$, so $(z, T\,()) = (F\,u,\; T\,())$.
